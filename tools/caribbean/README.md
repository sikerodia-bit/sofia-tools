# caribbean/

Tools built for Dominican Republic, Haiti, and the broader Caribbean region.
These tools work everywhere but are optimized for Caribbean context.

| Tool | Description | Network | Docker |
|------|-------------|---------|--------|
| currency_dop.py | USD ↔ DOP (Dominican Peso) exchange rates | Yes | No |
| currency_htg.py | USD ↔ HTG (Haitian Gourde) exchange rates | Yes | No |
| weather_caribbean.py | Caribbean weather (coming soon) | Yes | No |

## Contributing Caribbean Tools
Have a tool specific to DR, Haiti, or Caribbean?
We especially want:
- BHD/Popular/BanReservas banking tools
